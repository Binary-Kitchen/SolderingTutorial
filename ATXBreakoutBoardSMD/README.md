# ATX Breakout Board - DIP/SMD
A handy breakout board to get all important voltages from an standard ATX power supply.

<img src="manual/images/IMG_20180701_002328.jpg" width=400px alt="ATX Breakout Board">

- Status: **Complete**
- Difficulty: **4/5**
