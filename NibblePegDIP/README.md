# Nibble Peg - DIP
A very easy blinking peg with 4 LEDs.
A [nibble](https://en.wikipedia.org/wiki/Nibble) is a unit that consists of 4 bits (half a byte).

<img src="manual/images/IMG_5910.JPG" width=400px alt="Nibble Peg DIP">

- Status: **Complete**
- Difficulty: **1/5**
