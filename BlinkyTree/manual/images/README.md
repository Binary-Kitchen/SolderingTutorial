## Image Guidelines

### Technical Requirements
- **Format**: JPEG for photos, PNG for diagrams with transparency
- **Resolution**: Minimum 1920x1080 for detail shots, 400x300 for thumbnails
- **Quality**: High quality, well-lit photos with good contrast
- **Background**: Clean, neutral background for component photos

### Photography Tips
- Use good lighting (daylight or bright white LED)
- Ensure components are clearly visible and properly focused
- Show proper component orientation (especially for polarized parts)
- Include hands/tools in action shots to show scale and technique
- Take multiple angles for complex assembly steps

### File Naming
- Use descriptive names that match the manual text
- Include step numbers for sequential assembly images
- Use consistent naming convention throughout
- Keep filenames short but descriptive
